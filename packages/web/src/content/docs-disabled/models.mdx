This version uses the Claude SDK and supports Anthropic's Claude models.

---

## Available Models

The following Claude models are available:

| Model                        | Description                                |
| ---------------------------- | ------------------------------------------ |
| `claude-opus-4-5-20250929`   | Most capable model, best for complex tasks |
| `claude-sonnet-4-5-20250929` | Balanced performance and speed             |
| `claude-sonnet-4-20250514`   | Previous generation Sonnet                 |
| `claude-haiku-3-5-20241022`  | Fast and efficient for simple tasks        |

---

## Select a Model

Use the `/models` command to select a model:

```bash frame="none"
/models
```

---

## Set a Default

To set a default model, add it to your config:

```json title="opencode.json"
{
  "model": "anthropic/claude-sonnet-4-5-20250929"
}
```

The full ID format is `provider/model_id`. Since only Anthropic is available, all models use `anthropic/` prefix.

---

## Configure Models

You can configure model options in your config:

```jsonc title="opencode.jsonc"
{
  "provider": {
    "anthropic": {
      "models": {
        "claude-sonnet-4-5-20250929": {
          "options": {
            "thinking": {
              "type": "enabled",
              "budgetTokens": 16000,
            },
          },
        },
      },
    },
  },
}
```

---

## Variants

Claude models support thinking variants:

- `high` - High thinking budget (default)
- `max` - Maximum thinking budget

Select a variant using the format `model:variant`:

```
anthropic/claude-sonnet-4-5-20250929:max
```

---

## Custom Variants

You can define custom variants:

```jsonc title="opencode.jsonc"
{
  "provider": {
    "anthropic": {
      "models": {
        "claude-sonnet-4-5-20250929": {
          "variants": {
            "quick": {
              "thinking": {
                "type": "enabled",
                "budgetTokens": 4000,
              },
            },
            "deep": {
              "thinking": {
                "type": "enabled",
                "budgetTokens": 32000,
              },
            },
          },
        },
      },
    },
  },
}
```
