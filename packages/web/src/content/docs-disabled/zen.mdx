:::note
OpenCode Zen is not available in this fork. This fork operates independently without the OpenCode backend services.
:::

To use AI models with this fork, configure your providers directly using their native API keys:

- **Anthropic**: Get your API key from [console.anthropic.com](https://console.anthropic.com)
- **OpenAI**: Get your API key from [platform.openai.com](https://platform.openai.com)
- **Google**: Get your API key from [aistudio.google.com](https://aistudio.google.com)

Run the `/connect` command in the TUI and select your provider to configure your API key.
